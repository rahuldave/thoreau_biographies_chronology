# 1858-1862 Crisis Zoom Chronology

This focused chronology is a close-up view for the crowded interval in which
John Brown, Darwin's *Origin*, the Concord readers, Sanborn's arrest, Thoreau's
seed work, secession, war, and Thoreau's death overlap. It is intentionally
more detailed than the main book chronology while leaving the full aligned view
unchanged.

Date-check convention follows the main chronology. Every event includes a book
anchor when Fuller supplies the scene or interpretation and at least one web
source for the public event, person, publication, or historical frame.
